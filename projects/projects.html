System Admin/Documentation/Security

Fort Faux

Fort Faux, is a capstone project created and implemented by an assembled team of IT Systems Management and Security students with the Truro NSCC, to help execute the project to completion.

The team of four created a well-documented, mid-complexity, network infrastructure, for a faux educational centre. The objective of the project was to replicate an educational centres network infrastructure by setting up the hardware and software required.

The infrastructure relied on multiple servers using Window Server 2025, Security Onion and Wazuh for security monitoring, and Windows 11 for testing clients. The FF team then created multiple “Red team vs. Blue team” scenarios to assess the complexity of our faux educational centres security.

The project was a success, and we were able to create a secure network infrastructure that met the requirements of the project. The project provided valuable experience in system administration, documentation, and security, and it was a great opportunity for the team to apply their skills in a real-world setting.
