Windows Administration

AD2GO

AD2GO is a secure web application that allows authorized users to manage their company’s Active Directory (AD) directly from their phones.

Instead of being tied to a workstation, administrators can perform tasks such as creating user accounts, viewing inventory, and resetting passwords from any iOS or Android device.

On the technical side:

AD2GO uses Python and FastAPI for the backend, which connects to the target Active Directory through secure LDAP (LDAPS).

To protect remote access, the application is published through a Cloudflare Tunnel over HTTPS and uses JWT-based authentication.

The frontend is a Progressive Web App (PWA), AD2GO.ca, built with HTML, CSS, and JavaScript in Visual Studio Code.
